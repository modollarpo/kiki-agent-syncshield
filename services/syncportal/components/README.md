# SyncPortal Components

This directory contains production-ready, typed, and documented React components for the onboarding engine:

- `OnboardingChecklist.tsx`: Step-by-step onboarding checklist UI
- `LegalConsent.tsx`: Digital consent and legal agreement UI
- `OaasCalculator.tsx`: Outcome-as-a-Service revenue calculator

All components are enterprise-grade, styled for dark mode, and ready for integration.
