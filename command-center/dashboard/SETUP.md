# KIKI Command Center Dashboard

## Setup Instructions

1. Navigate to the dashboard directory:
   ```bash
   cd command-center/dashboard
   ```
2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```
3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Notes
- If you moved or renamed any imports, update all relative paths in your codebase to reflect the new location.
- If you use environment variables, copy your .env file here or set up new ones as needed.
- For production builds, use `npm run build` and `npm start`.
