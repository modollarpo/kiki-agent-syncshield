# KIKI Agent™ Design System

Welcome to the KIKI Agent™ UI/UX design system. This system ensures a high-trust, enterprise-grade, dark-mode experience for all Outcome-as-a-Service (OaaS) platform users.

## Brand Principles
- **High-Trust:** Deep navy backgrounds, emerald accents, and clear, modern typography.
- **Enterprise-Grade:** Accessible, responsive, and robust for mission-critical workflows.
- **Dark Mode:** All components use dark backgrounds and vibrant emerald highlights.

## Color Palette
- **Background:** #0f172a (Deep Navy)
- **Accent:** #10b981 (Emerald)
- **Surface:** #1e293b (Slate)
- **Text Primary:** #e0f2fe (Light Cyan)
- **Text Secondary:** #94a3b8 (Slate 400)

## Typography
- **Font:** Inter, sans-serif
- **Weights:** 400, 600, 700
- **Letter Spacing:** Tight for headers, normal for body

## Components
- **Modal:** For critical confirmations and onboarding
- **Table:** For enterprise data display
- **Notification:** For system and user feedback
- **Chart:** For quick, high-level metrics

## Accessibility
- All components are keyboard accessible
- Sufficient color contrast for WCAG AA
- Focus states and ARIA roles included

## Usage
Import components from `@kiki/ui` and use in your Next.js or React apps. See Storybook for live previews and props.
