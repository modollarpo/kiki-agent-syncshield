## Props

| Prop   | Type                                   | Description                        |
|--------|----------------------------------------|------------------------------------|
| logo   | `React.ReactNode`                      | Logo or brand element              |
| items  | `{ label: string; icon?: React.ReactNode; active?: boolean }[]` | Navigation items                   |
| footer | `React.ReactNode`                      | Footer content                     |

## Accessibility

- All navigation items are keyboard accessible.
- Active item is visually indicated.

## Theming

- Fully styled for dark mode using Tailwind CSS.
- Use `className` to extend or override styles.
