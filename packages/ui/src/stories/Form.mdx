# Form Component

The `Form` component provides a flexible, type-safe, and accessible way to build forms in your application. It supports text, email, password, number, select, checkbox, and textarea fields, with full dark mode styling and validation.

## Usage

```tsx
import { Form } from "@kiki/ui";

const fields = [
  { name: "email", label: "Email", type: "email", required: true },
  { name: "password", label: "Password", type: "password", required: true },
];

<Form
  fields={fields}
  onSubmit={(values) => console.log(values)}
  submitLabel="Sign In"
/>
```

## Props

| Prop         | Type                                 | Description                                      |
|--------------|--------------------------------------|--------------------------------------------------|
| fields       | `FormField[]`                        | Array of field definitions                       |
| onSubmit     | `(values: Record<string, any>) => void` | Callback for form submission                     |
| submitLabel  | `string`                             | Button label (default: "Submit")                 |
| initialValues| `Record<string, any>`                | Initial field values                             |
| loading      | `boolean`                            | Show loading state on submit button              |
| error        | `string`                             | Error message to display                         |
| className    | `string`                             | Additional CSS classes                           |

## Field Types

- `text`, `email`, `password`, `number`, `select`, `checkbox`, `textarea`

## Accessibility

- All fields are labeled and keyboard accessible.
- Required fields are indicated with a red asterisk.

## Example: Registration Form

```tsx
<Form
  fields={[
    { name: "name", label: "Name", type: "text", required: true },
    { name: "email", label: "Email", type: "email", required: true },
    { name: "role", label: "Role", type: "select", options: [ { label: "Admin", value: "admin" }, { label: "User", value: "user" } ], required: true },
    { name: "tos", label: "Accept Terms", type: "checkbox", required: true },
  ]}
  submitLabel="Register"
  onSubmit={(values) => console.log(values)}
/>
```

## Theming

- Fully styled for dark mode using Tailwind CSS.
- Use `className` to extend or override styles.
